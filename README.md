# BOT_Telegram

### Simples BOT telegram que listava algumas noticias e a cotação do dólar.

## Instalação e uso

### Rode comando abaixo para instalar as dependencias 

```
npm install
```

### Não esqueça de criar um arquivo de variaveis de ambiente com token do Telegram

```
.env
```

### Agora só rodar comando abaixo

```
npm run dev
```
ou
```
npm start
```
